# Contributing

Contributions should keep Cross-AI Review simple, portable, and useful for non-technical users.

Do not include API keys, tokens, credentials, personal data, client material, unpublished drafts, or confidential information. Use fictional examples only. Do not make claims that the workflow guarantees factual accuracy or prevents hallucinations. CLI guidance must rely on interactive provider authentication and must not bypass approval or sandbox protections.

Before opening a pull request, confirm that new templates preserve uncertainty, require evidence labels for material factual claims, and send high-stakes or ambiguous decisions to a human owner.
