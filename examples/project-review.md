# Project review packet — fictional example

## Mode
project-review

## Purpose and audience
Review the plan for a neighborhood book-swap event before assigning volunteers.

## Constraints
One afternoon event; no financial commitments in this draft.

## Draft or working material
The goal is to host a book-swap event next month. The event needs a venue, a table layout, a volunteer schedule, and a short public announcement. The group will measure success by attendance and the number of books exchanged. The event lead will confirm the venue before the announcement is published.

## Review instructions
Review independently. Check goals, scope, milestones, dependencies, risks, assumptions, ownership, and success measures. Do not invent missing details. Return the required review structure.
